# IELTS Learning

This project is a learning tool designed to help IELTS students improve their writing skills in the "Coherence and Cohesion" category for Task 2 essays.

## Features

*   **Lessons:** Explanations, examples, and banded samples (Band 5â†’8).
*   **Interactive Exercises:** Reordering sentences, improving paragraphs, choosing transitions, and rewriting for cohesion.
*   **Guided Essay Builder:** Using the Sentence Skeleton Framework.
*   **Automated Feedback:** Heuristic-based grading for paragraphing, connectors, referencing, and logical progression.
*   **Progress Tracking:** Monitor your improvement over time.

## Tech Stack

*   **Frontend:** React (with Vite) and Tailwind CSS
*   **Backend:** Python with FastAPI
*   **Database:** SQLite (for development)

## Getting Started

### Backend

1.  Navigate to the `backend` directory.
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the server: `uvicorn app.main:app --reload`

### Frontend

1.  Navigate to the `frontend` directory.
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`
